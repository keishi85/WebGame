ゲームのルール

    計算問題とクイズを正解して得点を稼ぐゲームです
    得点は回答するまでの速度と問題のレベル(果物の種類)によって変わります
      桃: 100 point
      りんご: 50 point
      みかん: 30 point
      レモン: 10 point
    5問計算問題を回答するとクイズが出題されるようになっています
    計算問題は下部の数値ボタンから答えを入力し、"Enter"を押してください

Game Rules

    This is a game where you earn points by correctly answering calculation problems and quizzes
    Scores vary depending on the speed of answering and the level of the question (type of fruit)
      Peach: 100 point
      Apple: 50 point
      Orange: 30 point
      Lemon: 10 point
    For calculation questions, enter the answer using the numerical buttons at the bottom and press "Enter"
    (There is no need to select a question when answering; the answer is correct if the input content corresponds to one of the questions on the screen)
